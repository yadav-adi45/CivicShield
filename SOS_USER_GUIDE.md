# 🆘 SOS Emergency System - User Guide

## Quick Start

### Accessing SOS
Look for the red **🆘 SOS** button in the top-right corner of the screen. It's always visible and pulses to draw attention.

---

## Step-by-Step Guide

### Step 1: Select Emergency Type

When you click the SOS button, you'll see three options:

```
┌─────────────────────────────────────┐
│  🚨 Police        Emergency: 100    │
│  🚑 Medical       Emergency: 108    │
│  ⚠️  Disaster      Emergency: 112    │
└─────────────────────────────────────┘
```

**Choose the appropriate emergency type:**
- **Police (100)**: Crime, theft, violence, law enforcement
- **Medical (108)**: Accidents, injuries, health emergencies
- **Disaster (112)**: Natural disasters, fires, major incidents

---

### Step 2: Location Sharing

The system will automatically request your location.

**If location is successful:**
```
✅ Location Acquired
📍 28.613939, 77.209021
Accuracy: ±15m
[View on Google Maps →]
```

**If location fails:**
```
⚠️ Location permission denied
[Try Again] [Continue Without Location]
```

**Tips:**
- Allow location access for faster response
- Location helps emergency services find you
- You can skip if location is unavailable

---

### Step 3: Describe Emergency

Provide details about your emergency:

**Text Input:**
```
┌─────────────────────────────────────────┐
│ E.g., "Car accident on NH-44,          │
│ 2 people injured" or "House fire,      │
│ smoke visible"                          │
│                                         │
│                          [🎤 Voice]     │
└─────────────────────────────────────────┘
```

**Voice Input (if supported):**
- Click the 🎤 Voice button
- Speak clearly
- Your speech will be converted to text

**This step is optional** - you can proceed without a description.

---

### Step 4: Take Action

Your emergency alert is ready! Choose an action:

```
┌──────────────────┬──────────────────┐
│  📞 Call         │  💬 Send SMS     │
│  Emergency       │                  │
│  100/108/112     │                  │
├──────────────────┼──────────────────┤
│  📤 Share        │  📋 Copy         │
│  Alert           │  Message         │
└──────────────────┴──────────────────┘
```

#### 📞 Call Emergency
- **Fastest option** for immediate help
- Directly dials emergency number
- Recommended for life-threatening situations

#### 💬 Send SMS
- Opens SMS app with pre-filled message
- Choose recipient (emergency contact, family)
- Good for notifying multiple people

#### 📤 Share Alert
- Share via WhatsApp, email, social media
- Reaches multiple contacts instantly
- Includes location and details

#### 📋 Copy Message
- Copies alert to clipboard
- Paste anywhere you need
- Useful for manual sharing

## Troubleshooting

### Location Not Working?
1. Check browser permissions (Settings → Privacy → Location)
2. Enable GPS on your device
3. Try refreshing the page
4. Use "Continue Without Location" option

### Voice Input Not Working?
1. Check microphone permissions
2. Use Chrome, Edge, or Safari (Firefox not supported)
3. Speak clearly and close to microphone
4. Use text input as alternative

### Share Button Not Working?
1. Use "Copy Message" instead
2. Manually paste in messaging apps
3. Try SMS option for direct sharing

### Call Button Not Working?
1. Manually dial emergency number
2. Check if phone app is installed
3. Try SMS option instead
